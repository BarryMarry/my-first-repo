## 📖 项目简介
**BarryHook** 是一个基于 LSPosed 框架的原创 Android Hook 项目，旨在通过系统级注入实现高效的性能优化与功能增强。

## ✨ 核心特性
- **自动化流水线**: 集成 GitHub Actions 实现 CI/CD 构建与测试。
- **标准架构**: 遵循 Android 模块开发规范，包含混淆规则与单元测试。
- **开源合规**: 采用 GPL-3.0 协议分发。

## 🛠️ 技术栈
- **Language**: Java 17
- **Framework**: LSPosed API / Xposed API
- **Automation**: GitHub Actions

- **v1.1.0**: 引入自动化构建脚本与 Release 标签。
